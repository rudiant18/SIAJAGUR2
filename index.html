📄 Sumber Data: Google Sheets

Cara menghubungkan database ke Google Sheets:
1. Buat Google Sheet dengan 2 sheet/tab: gejala dan penyakit.
2. Sheet gejala kolom: id, kategori, text (kategori: daun/batang/tongkol/umum).
3. Sheet penyakit kolom: id, nama, penyebab, gejala, penanganan — kolom gejala diisi id gejala dipisah koma (mis. g1,g5,g6), kolom penanganan diisi langkah dipisah dengan ;.
4. Klik File → Bagikan → Publikasikan ke web, pilih masing-masing sheet dengan format CSV, lalu salin link CSV yang dihasilkan untuk tiap sheet.
5. Tempel kedua link di bawah ini.
Data dari Sheets ini akan menggantikan data bawaan (gejala & penyakit) saat dimuat. Jika kosong atau gagal dimuat, sistem otomatis memakai data bawaan.

📝 Catat Hasil Diagnosa ke Google Sheets (Terpusat)

Setiap kali ada yang menekan "Diagnosa Sekarang" di tab Diagnosa, sistem dapat mengirim nama pengamat, gejala yang dipilih, dan hasil diagnosa ke 1 Google Sheet bersama — sehingga dosen/petugas bisa memantau seluruh laporan dari satu tempat.

Cara setup (sekali saja, oleh admin/dosen):
1. Buka script.google.com → Proyek baru.
2. Hapus kode bawaan, lalu tempel kode Apps Script yang sudah disiapkan (lihat tombol unduh di bawah).
3. Pada kode, ganti SPREADSHEET_ID dengan ID Google Sheet tujuan (lihat di URL sheet, di antara /d/ dan /edit).
4. Klik Deploy → New deployment → pilih jenis Web app → "Who has access" pilih Anyone → Deploy.
5. Salin URL Web App yang muncul (diakhiri /exec), tempel di kolom bawah ini.
Jika dikosongkan, hasil diagnosa hanya tersimpan lokal di tab Riwayat masing-masing pengguna.

Tentang SIJAGUR

Sistem Identifikasi Jagung — alat bantu deteksi dini penyakit jagung dari gejala lapangan.

Cara Kerja

Pengguna memilih gejala yang terlihat pada daun, batang, tongkol, dan pertumbuhan tanaman di tab Diagnosa. Sistem mencocokkan gejala tersebut dengan basis data penyakit, lalu menampilkan persentase kecocokan beserta cara penanganannya. Persentase dihitung dari jumlah gejala yang cocok dibagi total gejala khas penyakit tersebut.

Sumber Data

Data dasar gejala dan penyakit dapat dimuat langsung dari Google Sheets publik yang dikelola dosen atau petani melalui tab Data Penyakit, sehingga basis pengetahuan sistem dapat diperbarui tanpa mengubah kode. Jika sumber Google Sheets belum diatur atau tidak dapat diakses, sistem memakai data bawaan sebagai cadangan.

Kelola Data Tersimpan

Data tambahan lokal (gejala baru, penyakit baru, dan riwayat diagnosa) tersimpan secara pribadi di perangkatmu dan tidak terlihat oleh pengguna lain. Pengaturan sumber Google Sheets juga tersimpan lokal per perangkat.
